🏀 SLP Pool Rules

Roster Selection

A dollar value is assigned to each Division I college basketball team. You select a roster of 11 schools, the total value of which cannot exceed $100. Your roster is locked for the entire season once submitted.

You must select at least one school from each of the following conferences:

ACC · Big East · Big Ten · Big 12 · SEC

The remaining 6 schools (wildcards) can be any Division I school. Only the top 4 scoring wildcards count toward your final score — the lowest wildcard is dropped.

Entry Fee

Entry fee: $75 per entry.

Regular Season Scoring

Non-conference win+1 pt
Conference win+2 pts
Conference regular season title (best winning %)+5 pts
Tie for conference title (2-way)+3 pts each
Tie for conference title (3+ way)+2 pts each

Conference Tournament Scoring

Conference tournament win+2 pts
Conference tournament title+5 pts

NCAA Tournament Scoring

#1 seed in Field of 64+8 pts
#2–4 seed in Field of 64+6 pts
#5–8 seed in Field of 64+4 pts
#9–16 seed in Field of 64+2 pts
Round of 64 win+2 pts
Round of 32 win+4 pts
Sweet 16 win+6 pts
Elite 8 win+8 pts
Final Four win+10 pts
Championship game win+12 pts

Notes

Conference game designations follow each conference's official rules. The non-Big Dance post-season tournaments do not count. If a team cannot participate in the NCAA Tournament due to reclassification, they still receive conference tournament points but only receive seeding points for their actual bid.

⚙️ How to Update the Website

All scores and team values live in data.json. Edit that file to update the site — no other files need to change.

1. Set Team Values

Once you've assigned dollar values for the 2025–26 season, open data.json and update the "value" field for each team in each entry's "picks" and "wildcards" arrays. Also update "totalValue" for each entry.

"picks": { "ACC": { "team": "Duke", "value": 16, "points": 0 }, "Big East": { "team": "UConn", "value": 15, "points": 0 }, ... }, "wildcards": [ { "team": "Gonzaga", "value": 13, "points": 0, "counts": true }, ... ]

2. Update Scores

To update scores manually, edit each entry's "score" field and the "points" for individual teams. Also update "lastUpdated" at the top of the file.

{ "name": "Connor Pelton", "totalValue": 98, "score": 42, ← update this "picks": { ... } }

3. Auto-Update with ESPN + Claude Code

🏀 This is the best workflow: Use Claude Code to run a scoring script that pulls from ESPN and updates data.json automatically.

The ESPN public API provides live college basketball scores and standings. Ask Claude Code to write a script that:

• Fetches game results from the ESPN API

• Applies SLP scoring rules (1pt non-conf win, 2pt conf win, 5pt conf title, etc.)

• Calculates each entry's score based on their roster

• Writes updated scores back to data.json

• Can be scheduled to run daily (e.g., every morning)

# Ask Claude Code something like: # "Write a Python script that pulls college basketball # scores from ESPN's API, applies the SLP scoring rules # defined in data.json, and updates each entry's score # in data.json. Then commit and push to GitHub."

Once the script is set up, Claude Code can run it on a schedule. Every push to GitHub automatically updates the live site.

4. Mark Wildcards as Dropped

At the end of the season, set "counts": false on the lowest-scoring wildcard for each entry. The site will automatically gray it out and exclude it.
